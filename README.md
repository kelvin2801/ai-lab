# ai-lab

## Included Scripts

### `scripts/validate_credit_cards.py`
A script to validate credit card numbers using the Luhn algorithm. It supports taking numbers as command-line arguments or via a file input.

**Usage:**
```bash
python scripts/validate_credit_cards.py 1234567890123456
# or with a file
python scripts/validate_credit_cards.py -f cards.txt
```
